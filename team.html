Team coordination

Team work is organised using the bitcoin-multibit mailing list. The MultiBit team lead is the user "Jim Burton" in this group. You can also contact Jim by email at:

"jim" followed by an "at" sign followed by "multibit.org".

or at the #multibit channel at irc.freenode.net

MultiBit thread on bitcointalk.org

Most announcements and MultiBit related news are posted on this thread on bitcointalk.org

Source code

All code is open source and available at github.com.

The read-only repository is available at: git://github.com/jim618/multibit.git

The continuous integration build machine is ci.xeiam.com

Translating MultiBit

MultiBit is currently being translated. If you would like to help, visit our translation site.
